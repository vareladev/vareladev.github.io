Módulo 3. Visualización y análisis en Power BI

Guía de aprendizaje 1. Análisis de datos básico con Power BI

Indicaciones.

Seguir los pasos y resolver los desafíos presentados en esta guía de aprendizaje. Asegurarse de guardar los avances realizados en cada paso. Al finalizar, se debe subir los recursos solicitados en la plataforma e-campus.

Objetivos de aprendizaje.

Que el estudiante pueda:

  1. Familiarizarse con el entorno de Power BI.
  2. Aprender a utilizar las herramientas de visualización básicas que se encuentran disponibles en Power BI, para crear gráficos que permitan mostrar los datos de forma eficiente.
  3. Desarrollar la capacidad lógica necesaria para poder responder interrogantes que permitan tomar decisiones relevantes.

Formato de entrega de la actividad

Para que la entrega de esta actividad sea catalogada como válida, se deben seguir las siguientes indicaciones.

  1. Entregar el archivo de power BI. Archivo en formato .pbix que contiene todas las visualizaciones construidas.
  2. Construir una memoria de trabajo. Documento en formato .pdf que incluye todas las preguntas (desde la pregunta 1 a las pregunta 7, incluyendo las preguntas secundarias), con su respectiva respuesta. Se sugiere incluir una captura de pantalla con la visualización (gráfico) utilizada para realizar el análisis que ha permitido responder a cada pregunta. En la evaluación se valorará el formato y la formalidad en la memoria de trabajo.
  3. Entrega. Una vez finalizada la actividad, se deberá subir a la platanforma e-campus los productos solicitados.
  4. Modalidad de trabajo. Esta actividad esta planificada para poder realizar en formato individual, sin embargo, esta permitido trabajar en equipos de 2-3 personas. En este caso, un representante del equipo subirá los productos solicitados a la plataforma e-campus.

Power BI es un servicio de análisis empresarial de Microsoft que permite conectar y visualizar datos de forma interactiva a través de una plataforma unificada. Es una de las herramientas de modelado de datos más utilizadas en la actualidad. De hecho, el uso de Power BI, especialmente la versión de escritorio Power BI o Power BI Desktop, permite crear modelos de datos e informes efectivos con los distintos servicios de Power BI gracias a sus opciones de soporte de autoservicio.

Requerimientos

En la lista siguiente, se proporcionan los requisitos mínimos para ejecutar Power BI Desktop:

Requerimiento

Descripción

Sistema operativo

Windows 10, 11, Windows Server 2016 R2 o posterior.

Memoria RAM

Al menos 2 GB disponible; se recomienda 4 GB o más.

Pantalla

Se necesita al menos 1440x900 o 1600x900 (16:9). No se admiten resoluciones inferiores a 1024x768 o 1280x800, ya que ciertos controles (por ejemplo, para cerrar las pantallas de inicio) solo se muestran con resoluciones superiores a estas.

CPU

Se recomienda un procesador de 64 bits (x64) a 1 gigahercio (GHz) o superior.

Otras configuraciones de Windows

Si configura la pantalla para cambiar el tamaño del texto, las aplicaciones u otros elementos en más del 100 %, no verá algunos cuadros de diálogo con los que debe interactuar para seguir usando Power BI Desktop. Si se produce este problema, compruebe la configuración de la pantalla en Configuración>Sistema>Pantalla en Windows y use el control deslizante para devolver la configuración de pantalla al 100 %.

Otro software necesario

Se requiere de WebView2, que se suele instalar junto con Power BI.
.NET 4.7.2 o posterior.

Instalando Power BI

1. Ir al sitio oficial de Power BI: Descargar Power BI

2. Ubicar la sección de "Microsoft Power BI Desktop" y hacer clic en "Descargar".

3. Aceptar el mensaje que comunica que se abrirá la aplicación de Microsoft Store.

4. En Microsoft Store, hacer clic en "Obtener" y esperar a que la descarga finalice. (Se descargarán aproximadamente 750MB). Nota: En este paso, Microsoft Store solicitará iniciar sesión con una cuenta de correo de outlook o hotmail.

5. Una vez que el software se ha descargado, se instalará y si no surgen problemas en el proceso, Power BI Desktop estará disponible en nuestra computadora.

Antes de iniciar el proceso de análisis de datos, existen pasos importantes que deben ser gestionados de forma correcta para minimizar posibles errores e inconsistencias. Uno de ellos es dar el formato adecuado a los datos.

8. Ir al panel "vista de tabla", ubicado en la barra de herramientas lateral izquierda y esperar a que se carguen los datos en formato de tabla.

9. Seleccionar la columna "fecha". En la barra de herramientas se habilitará la sección "Herramientas de columna". Seleccionar esta opción y ubicar la sección formato. En esta opción cambiar el formato de la fecha a "(d/m/yyyy)".

10. Cambiar el formato de las columnas "PrecioUnitario" y "TotalVentas" a "Moneda". Si es necesario, cambiar el símbolo de moneda en las opciones que aparecen en la sección de formato. En este ejemplo se utilizará el símbolo "$".

Pregunta 1. ¿Cuál fue el mes con la mayor cantidad de ventas utilizando como criterio las unidades de productos vendidas?

Para poder realizar el análisis, se sugiere identificar las entidades que se esperan relacionar. En primer lugar, para definir si tenemos disponible los datos necesarios para responder a las interrogantes, y, en segundo lugar, si los datos tienen el formato adecuado o habrá necesidad de hacer algún proceso de transformación. Para la pregunta 1, se solicitan dos entidades: el tiempo (porque se pregunta por un mes), y las unidades de productos vendidas. Si se observa el banco de datos, se encuentran disponibles ambas variables. Por lo que se procede a crear una visualización de los datos adecuada que permita responder a la pregunta inicial.

1. Ir al panel "vista de informe", ubicado en la barra de herramientas lateral izquierda.

2. Seleccionar un gráfico de líneas. Nota. El grafico de líneas es ideal para analizar datos continuos, por ejemplo, la variable tiempo.

3. Asegurarse de que el gráfico de líneas este seleccionado, ubicar en el panel de visualizaciones a los campos "Eje X" y "Eje Y".

4. Seleccionar la columna "Fecha" del banco de datos y arrastrar hasta el eje X.

5. Seleccionar la columna "CantidadVendida" del banco de datos y arrastrar hasta el eje Y.

6. El gráfico de líneas debería aparecer mostrando los años, pero no es muy representativo, por lo que se debe ir al siguiente nivel de jerarquía de fechas. Para ello, hacer clic en el botón "Ir al siguiente nivel de jerarquía" para visualizar los datos a nivel de trimestres. Si se hace clic nuevamente, se podrán visualizar los datos a nivel de meses.

7. Ahora que se dispone de la visualización adecuada, es posible responder a la primera pregunta.
